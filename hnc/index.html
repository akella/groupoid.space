System F Intermediate Language

System F Core. HNC contains Minimal nonrecursive core with Damas-Hindler-Milner type inference. This type system is usefull for code generation to such languages as C++, Java, C#, ML.

Readable Extraction. The main feature of HNC is readable targets. I.e. C++ generated code preservse variable naming and is editable by target language developers. So you can extend generated code withing your target language.
