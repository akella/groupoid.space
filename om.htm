Rules

Lambda Cube Top configuration with Dependent Types.
Note that term is star and type is box and also
that TypeN-1 inhabits TypeN. Terms inhabit Type0 which is Prop. 

  rule(star,star) -> star;
  rule(box,star)  -> star;
  rule(star,box)  -> box;
  rule(box,box)   -> box.
