Bundle package contains axiom of infinitesimal comonadic shape modality $\Im$.

Definition (Infinitesimal Shape Modality). The two maps $\Im$∶ U -> U and $\Im$Unit (A: U)∶ A -> Im A are called shape modality if i) $\Im$-Unit is an equivalence; ii) the type A is called coreduced. Identity types of coreduced types are coreduced; iii) if B∶ $\Im$ A -> U is a dependent type such that for all a∶ $\Im$ A the type B(a) is coreduced, then we can define a section of B by induction.
